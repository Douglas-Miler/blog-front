A CODE UP

No ano de 2018, cinco estudantes de Ciência da Computação decidiram se juntar, para fazer o que ia ser um dos melhores TCCs já visto na Universidade São Judas Tadeu. Este Trabalho de Conclusão de Curso incluia a tarefa de abrir uma empresa fictícia, apenas para lançar o projeto que haveeria de ser construído. Nasce então a Code UP, uma empresa "de mentirinha", um CNPJ falso.
Voltada para o ramo de desenvolvimento de games, a Code UP adquiriu seu primeiro elogio da crítica (colegas de sala e o professor orientador) quando lançou IA 2100, um game de ficção científica 2D para desktop. Embora o trabalho árduo em sua construção, o IA 2100 nos custou a tão esperada nota 10!
A Code UP está há dois anos no mercado, embora esteja em recesso desde a conclusão da faculdade, em 2018. Seus integrantes nunca mais se encontraram, e um deles, Gustavo Ortiz, nunca mais foi visto, desde a apresentação do TCC...
